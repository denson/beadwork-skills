Beat 1 · The hook

Agents make mistakes. Chat history can't help you find them.

In any non-trivial agent workflow, agents make mistakes — they drift on a convention they were supposed to follow, they apply different criteria across similar tasks, they miss things a peer just flagged.

Same work. Different substrate. Only one is auditable.

The cheap way to catch those mistakes is another agent reading the work — not the chat that produced it, but the actual decisions and verdicts. Chat history doesn't work for that: unstructured stream-of-consciousness, speaker is implicit, timing is muddied, mostly noise.

bw (beadwork) is structured: every comment records who wrote it, when, on which ticket, in what shape. An audit agent reading bw show gets the substrate it needs to actually find the mistakes.

The structural property: signed + timestamped + structured. That's what makes meta-analysis tractable.

Chat doesn't have that shape. bw does.
