Where to go from here.

This was the overview. Five companion skills go deeper on specific use-case angles, plus an install skill that drives setup end-to-end.

Five companion skills.

  • beadwork-as-memory — bw as durable agent memory. The "future-you reads bw show" pattern across long-running work, single-agent and multi-agent.
  • beadwork-as-bus — bw as agent-to-agent communication. Multi-agent pipelines, cross-team coordination, cross-organization handoffs.
  • beadwork-for-meta-analysis — running an agent over the bw history of other agents to audit their work, check methodology, or synthesize across many earlier sessions. The structural advantage over chat history.
  • beadwork-for-decisions — bw as decision log / ADR / compliance audit trail. The "defensible by construction" angle.
  • beadwork-install — drive the install end-to-end. Claude can run it for you; the only choice you make is the storage mode (local-only, private GitHub, or public GitHub with the privacy implications spelled out).
